Time series classification

Published: Monday, January 1, 1

One important aspect of smart infrastructure research and development consists of automatic classification of time series data. Our recent work using shallow and deep machine learning methods for appliance classification in smart grids is documented in two recent papers and the code is available also as a CaaS service.
